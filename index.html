My Journal & Pensieve

One-time setup — takes about 5 minutes

Step 1 — Create a Google Cloud project

Go to console.cloud.google.com, sign in, and create a new project (name it anything, e.g. "My Journal").

Step 2 — Enable the Google Drive API

In your project, go to APIs & Services → Library, search for Google Drive API and click Enable.

Step 3 — Create OAuth credentials

Go to APIs & Services → Credentials → Create Credentials → OAuth client ID.

Choose Web application. Under Authorised JavaScript origins, add the URL you'll host this file at (e.g. https://yourname.github.io or http://localhost for local use).

Also add it under Authorised redirect URIs.

Copy your Client ID and paste it here:

It looks like: 123456789-abcdefg.apps.googleusercontent.com

Step 4 — Configure OAuth consent screen

Go to APIs & Services → OAuth consent screen. Set User Type to External, fill in the app name (e.g. "My Journal"), and add your own email as a test user. You only need the drive.file scope.

Ready to go

Once you've pasted your Client ID above, click Start. You'll be prompted to sign in to Google once, then your journal opens.

Your Client ID is saved only in this browser. Nothing leaves your device.
